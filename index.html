Sistema de inventario

¿Qué es un sistema de inventario?

Es un sistema de seguimiento que informa sobre la cantidad de materias primas, suministros o productos terminados que se tienen disponibles. Cuando la gente piensa en un sistema de inventario, es común relacionarlo con la industria minorista. Sin embargo, aunque las tiendas requieren un sistema de inventarios eficaz para poder operar con éxito, este también se usa en muchos otros tipos de negocios, como manufactura, servicios públicos, salud, gobierno y educación.
